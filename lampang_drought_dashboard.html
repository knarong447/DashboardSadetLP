# จังหวัด ดัชนี ระดับ ภาค
การประเมินสถานการณ์ — พื้นที่บ้านเสด็จ ตำบลพิชัย อำเภอเมืองลำปาง
ดัชนีภัยแล้ง 34 คะแนน อยู่ในระดับเฝ้าระวัง ใกล้เคียงค่าเฉลี่ยภาคเหนือ (35) — คาดการณ์ El Niño ปี 2569 จะทำให้ภาคเหนือมีฝนน้อยกว่าปกติ 20–40% แนะนำเร่งดำเนินการ 3 มาตรการเร่งด่วน: (1) สำรองน้ำชุมชน 75 ครัวเรือน (2) ติดตั้ง IoT Station เฝ้าระวัง (3) อบรม ADM 25 คนก่อนเดือนมิถุนายน 2569
แหล่งข้อมูล
ข้อมูลดัชนีภัยแล้งดึงจาก GISTDA CropsDrought API (cropsdrought.gistda.or.th/api/rest/zonal/province) ร่วมกับ กรมชลประทาน, สทนช. และกระทรวงเกษตรฯ — เปิดให้ใช้งานโดยไม่ต้อง API Key
โครงการ Pre-SRA เป้าหมายข้อ 6 | มหาวิทยาลัยราชภัฏลำปาง | ทุน บพท. ปี 2568–2570
